WORK Selected builds

Four projects, chosen for depth over breadth — each one moved from a real constraint through failure to a working system.

PRJ — ESP·MESH·SENSE

Smart-Home Environmental Sensing

Problem: single-point sensors miss how air quality and temperature actually vary across a home.

Built a mesh of ESP32 nodes over ESP-NOW, feeding an EdgeImpulse-trained model to classify room conditions in real time. Final-year project, University of Manchester.

  • ESP32
  • ESP-NOW
  • Edge ML
  • C++

PRJ — FPV·FC·ESC

FPV Drone Flight Stack

Problem: off-the-shelf FC/ESC combos leave little room to tune for a specific frame and flight style.

Hand-built flight controller and ESC configuration with DShot protocol and BLHeli firmware tuning, iterating through several crashes to a stable setup.

  • DShot
  • BLHeli
  • PCB design
  • Firmware

PRJ — STM32·TRACK

Line-Tracking Buggy

Problem: closing the loop between sensing and motor response fast enough to hold a line at speed.

STM32-based buggy with IR line sensing and a tuned PID control loop, built to hold a track reliably at increasing speeds.

  • STM32
  • PID control
  • C

PRJ — UI·ENCODER

Dual Rotary Encoder Display UI

Problem: physical dials feel more precise than touchscreens for fast, repeated adjustments — most embedded UIs ignore that.

Two-encoder input system driving a round GC9A01 display, including a ported RoboEyes animation layer for state feedback.

  • GC9A01
  • Embedded UI
  • C++
